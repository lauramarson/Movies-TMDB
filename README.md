# Movies-TMDB
An app about movies that consumes TMDB API, where you can search for movies, see the details like synopsis and genres, and add them to a favorite list.

## Technologies used:
- Swift
- UIKit
- API
- Core Data
- CocoaPods
- Alamofire
- SDWebImage
- Codable
- Storyboard, Autolayout
- TableView, CollectionView
